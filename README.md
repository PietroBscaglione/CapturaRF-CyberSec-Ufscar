# **Captura RF – Desafio da Disciplina Segurança Cibernética (UFSCar – 2025/2)**

Este repositório contém as soluções, análises e materiais desenvolvidos para o **Desafio de Captura RF**, realizado no contexto da disciplina de **Segurança Cibernética** na Universidade Federal de São Carlos (UFSCar) no semestre **2025/2**.  
O desafio envolveu a interceptação, decodificação e interpretação de sinais **RF (Radio Frequency)** utilizando ferramentas de SDR (Software Defined Radio) e técnicas de engenharia reversa de protocolos de comunicação.
